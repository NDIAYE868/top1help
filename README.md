# 🎓 TOP1HELP

**TOP1HELP** est une plateforme numérique intelligente d'orientation vers la formation, la certification et l'insertion professionnelle. Elle centralise les opportunités et accompagne les étudiants, diplômés et chercheurs d'emploi via un moteur de recommandation basé sur l'Intelligence Artificielle.

---

## 📝 Informations Générales
*   **Porteur de projet** : Serigne Arona SENE
*   **Contexte** : Master 1 Intelligence Artificielle & Data Science (M1 IA & DS) - Swiss UMEF University Campus de Dakar (2025-2026)
*   **Version** : 1.0
*   **Technologies principales** : React.js, React Native (ou Flutter), FastAPI (Python), PostgreSQL, TensorFlow, Scikit-learn, LangChain, API OpenAI.
